HTML & CSS Cursus

Grafische en Digitale Media

Welkom bij de HTML & CSS cursus van AP Hogeschool Antwerpen. In deze cursus leer je de fundamenten van webontwikkeling: van basis HTML structuur tot geavanceerde CSS styling en responsive design.

Wat ga je leren?

Deze cursus behandelt HTML documentstructuur, semantische elementen, CSS styling, layout technieken zoals Flexbox en Grid, en responsive design principes. Je leert door te experimenteren en te bouwen.

Hoe werkt het?

Gebruik de navigatie om door de verschillende lesweken te bladeren. Elke week bouwt voort op de vorige. Experimenteer met de code, breek dingen, en leer van je fouten!

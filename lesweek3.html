Lesweek 3: CSS Box Model & Layout

Ruimte en positionering

Het CSS Box Model beschrijft hoe elk element bestaat uit content, padding, border en margin. Begrip hiervan is essentieel voor het creëren van goede layouts.

Box Model Componenten

Content is de inhoud zelf, padding is ruimte binnen de border, border omringt de padding, en margin creëert ruimte tussen elementen.

Layout Technieken

Moderne CSS biedt krachtige layout tools zoals Flexbox voor één-dimensionale layouts en CSS Grid voor twee-dimensionale layouts.
